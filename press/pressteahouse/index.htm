Discretelogics releases TeaHouse

Vienna, Austria (March 4, 2013) - Discretelogics announces TeaHouse, a Windows application to browse and visualize time series data like financial, sensor or computer performance metrics data.

TeaHouse complements TeaFiles, the free file format for fast and simple time series storage. Designed for high performance data access, files of Gigabyte size are opened and displayed within Milliseconds. Arbitrary time series formats are visualized by Points, Lines, Histograms, Areas, Bands, Bars and Candles. Charts update automatically whenever files change, which allows running TeaHouse beside other analysis tools.

TeaHouse is available as a standalone windows application and as extension to Microsoft Visual Studio. Based on Microsoft Visual Studio, TeaHouse can be extended by VSIX PLugins. A Yahoo Finance Data Download extension is freely available.
